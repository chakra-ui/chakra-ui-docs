The Code component is a single part component. Read more about how to style
multipart components in the
[style system docs](/docs/styled-system/component-style#styling-single-part-components)

more information in the source for the styles
[here](https://github.com/chakra-ui/chakra-ui/blob/main/packages/components/theme/src/components/code.ts)

## Example

```jsx live=false
import { extendTheme } from '@chakra-ui/react'

const theme = extendTheme({
  components: {
    Button: {
      baseStyle: {
        color: "green.500",
        fontWeight: "bold",
        borderRadius: "xl",
        _hover: {
          borderRadius: "md",
        },
      },
    },
  },
});
```
